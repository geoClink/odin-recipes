# Odin Recipes

A basic recipe website built with HTML as part of The Odin Project Foundations curriculum.

## Live Demo

[View on GitHub Pages](https://geoclink.github.io/odin-recipes/
)

## Features

- Main index page with links to individual recipes
- Individual recipe pages with ingredients and instructions
- Images for each recipe
- Proper HTML structure and semantic elements

## Technologies

- HTML
- CSS

## What I Learned

- HTML document structure and boilerplate
- Working with links and navigation between pages
- Using lists for ingredients and instructions
- Adding images with proper alt attributes
- Organizing a multi-page project
